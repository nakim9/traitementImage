-> Programme de traitement d'image
Copie l'image hiboux.bmp dans une image copie.bmp
L'image est séparée en 3 matrices RGB avant d'etre reconstituée a l'identique

-- How to use --
Compiler avec la commande : make
executer avec la commande : ./exec

-- What's done --
Q2 et Q3 : créer3Matrice RGB et créerImage dans image.c et image.h
